Summer Readings

10. mai 2018

Originally debuting at Undercover’s Paris Fashion Week show, the Undercover x Nike React Element 87 is set to release in five colorways. The sneaker features a translucent upper, exterior taping on the heel, trail running style laces, an exposed lace collar and, of course, a React sole. If that wasn’t enough to get Takahashi enthusiasts excited, the React Element 87 also features Undercover branding on the upper.
